async def init_db(database_url: str) -> None:
    global _pool
    _pool = await asyncpg.create_pool(
        dsn=database_url,

async def close_db() -> None:
    global _pool
    if _pool:

**Migrated functions** — all remove `async with _lock:` wrapper, use `_pool` methods directly:

`log_signal(...)` -> Use `_pool.fetchval()` with `RETURNING id`. Parameters use `$1, $2, ...` up to `$10`. Pass `datetime.now(timezone.utc)` for timestamp instead of `.isoformat()` — asyncpg handles datetime objects natively.

`log_trade(...)` -> Same pattern. Use `$1` through `$12`. `RETURNING id` with `fetchval()`. Note: ticket parameter is `int` — asyncpg handles BIGINT mapping automatically.

`update_trade_close(...)` -> Use `_pool.execute()`. Parameters: `$1` pnl, `$2` close_price, `$3` datetime.now(timezone.utc), `$4` ticket, `$5` account_name.

`get_daily_stat(account_name, field)` -> Call `_validate_field(field)` first. Use `_utc_today()`. Use `_pool.fetchval()` with `f"SELECT {safe_field} FROM daily_stats WHERE date=$1 AND account_name=$2"`. Return `result if result is not None else 0`.

`log_pending_order(...)` -> Use `fetchval()` with `RETURNING id`. Change `expires_at` parameter from `str` to `datetime` — callers already pass ISO strings but asyncpg needs datetime objects. Accept both str and datetime: if str, parse with `datetime.fromisoformat(expires_at)` for backward compat. Use `NOW()` for created_at instead of passing it.

`get_expired_pending_orders()` -> Use `_pool.fetch()`. Pass `datetime.now(timezone.utc)` as parameter. Convert rows with `[dict(r) for r in rows]`.

`mark_pending_cancelled(order_id)` -> Use `_pool.execute()`. `$1` for order_id.

`mark_pending_filled(ticket, account_name)` -> Use `_pool.execute()`. `$1` ticket, `$2` account_name.

`get_recent_trades(limit=50)` -> Use `_pool.fetch()`. `$1` for limit.

`get_recent_signals(limit=50)` -> Use `_pool.fetch()`. `$1` for limit.
