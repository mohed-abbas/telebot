## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-08-06 | Spoofing | POST /auth/login brute force | mitigate | argon2 slow hash (reused) + per-IP 5/15min lockout (`db.get_failed_login_count`) returned as 429; nginx `limit_req zone=telebot_login` extended to `/api/v2/auth/login` (deploy note, D-14). |
| T-08-07 | Spoofing/Tampering | CSRF on /api/v2 mutations | mitigate | `telebot_csrf` double-submit + `X-CSRF-Token` + `compare_digest`; 403 on mismatch, proven by tests/test_api_csrf.py (D-16 hard gate). |

| T-08-09 | Information Disclosure | telebot_csrf readable cookie | accept | Grants nothing alone (not a session credential); session cookie stays httpOnly (V3). |
